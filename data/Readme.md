# Data files

## Instances

Instances have a ".txt" extension.

Their format is:
```txt
[Number of vertices]
[For every vertex] {
[Demand]
}
[Number of edges]
[For every arc] {
[Vertice 1] [Vertice 2] [Capacity] [Costs]
}
```

## Solutions

Solutions have a ".sol" extension.

Their format is:
```txt
[For every edge] {    
[Flow passing by the edge]
}
```
